BonsaiPress

The lean multi-client CMS.
No database. Just files, Git, and speed.

What makes BonsaiPress different

🌳

No Database

site_structure.xml is the single source of truth. No MySQL. No migrations. No runtime dependencies.

⚡

Static Export

One command generates pure HTML. Blazing fast, zero PHP at runtime — just files on a web server.

🔀

Multi-Client

One CMS engine, unlimited client projects. Each project is its own Git repo. Switch in seconds.

🚀

Hash-Diff Deploy

Only changed files are uploaded via FTPS. No full syncs. No downtime. No guesswork.

The entire workflow

bonsai start               # Docker up: CMS :8080, Preview :8081, Watcher :8001
bonsai new myclient        # New project from template, git-ready
bonsai static              # Generate pure static HTML into current/static/
bonsai deploy              # Upload only what changed — hash-diff via FTPS
bonsai switch otherclient  # Swap project instantly
